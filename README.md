# Node.js with Express, Sequelize and MySQL

## Project setup

```
npm install
```

### Run

```
node server.js
```

# Project use

- Squelize (ORM) - version 6.5.1
- Nodejs - version 14.15.4
- Express - version 4.17.1
- Body Parser - version 1.19.0
- CORS - version 2.8.5
- MYSQL2 - 2.2.5
